# Bazquiat Website

A clean modern clothing business website built with simple web technologies:
- HTML for structure
- CSS for styling
- JavaScript for interactivity

## Files
- `index.html` - page content and sections
- `style.css` - layout, colors, spacing, and responsive design
- `script.js` - shop items, cart behavior, and order form logic

## Features
- Homepage hero section
- Pieces of the Day section
- About section with brand history
- Purchasing section with clothing pieces
- Cart with add/remove actions
- Ordering form
- White and gold premium color palette
- Mobile-friendly layout

## How to use
Open `index.html` in a browser.
